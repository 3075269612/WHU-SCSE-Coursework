# 《人工智能》课内实验

本目录整理人工智能课程课内实验代码。当前文件结构已经比较清晰：根目录按实验编号放置可直接阅读和运行的 Python 脚本，配套资料、数据集、工具脚本和运行输出分别归入独立目录。

## 内容结构

- `2.3.*_*.py`：状态空间搜索与经典问题求解，包括八数码、传教士与野人、猴子和香蕉、汉诺塔、01 背包和旅行商问题。
- `3.3.*_*.py`：谓词逻辑、规则表示与归结推理相关实验。
- `4.3.*_*.py`：遗传算法、蚁群算法和模拟退火求解 TSP。
- `5.3.*_*.py`：鸢尾花数据集预处理及 KNN、决策树、朴素贝叶斯、支持向量机分类。
- `6.3.*_*.py`：PCA 降维、人脸识别和 K-means 聚类分析。
- `data/`：实验所需数据集与约束文件，例如 `S.txt`、`secom.data`、`testSet.txt`、`ORL/`。
- `docs/`：课程实验说明 PDF。
- `tools/`：数据集准备等辅助脚本。
- `output/`：实验运行生成或保留的结果图。

## 运行提示

建议在本目录下创建本地 Python 虚拟环境后运行实验脚本。第 5、6 章实验常用依赖包括 `numpy`、`pandas`、`scikit-learn`、`matplotlib`、`opencv-python` 等。

部分第 6 章实验依赖外部数据集，可先运行：

```bash
python tools/setup_datasets.py
```

本地环境目录、编辑器配置和 Python 缓存文件不纳入版本管理；保留在仓库中的主要是课程代码、必要数据、实验资料与可复查的输出结果。

# WHU-SCSE-Coursework

武汉大学国家网络安全学院（网络空间安全）相关课程实验、课程项目与阶段性作业归档。

本仓库按课程或项目拆分目录，尽量保留可运行代码、必要数据、实验说明、结果图表与报告文件，便于后续复盘、查阅和迁移。部分体量较大或独立维护的项目以 Git submodule 形式接入。

## 目录概览

| 目录 | 内容 |
| --- | --- |
| `《人工智能》课内实验/` | 人工智能课程课内实验，覆盖搜索、约束推理、智能优化、分类、降维、聚类、序列模型与图神经网络等主题。 |
| `《人工智能》课外实验（要求自学）/` | 人工智能课外自学实验，当前以 submodule 形式维护。 |
| `《人工智能》期末大作业/` | 短信垃圾邮件分类项目，包含数据处理、模型训练、评估结果、图表与报告。 |
| `《密码学引论》实验大作业/` | 基于 SM4-CBC-CTS、RSA-OAEP、GTK 3 与 OpenSSL 的文件加密系统实验。 |
| `《程序设计》实验大作业/` | 程序设计课程大作业相关代码、素材与文档。 |
| `《自然语言处理》实验/` | 自然语言处理课程实验，覆盖词向量、文本分类、句法分析、实体识别、预训练模型与大模型应用。 |
| `第一学年小学期项目/` | 第一学年小学期项目代码与配套资源。 |
| `第二学年小学期项目（数据爬取）/` | 数据爬取小学期项目，当前以 submodule 形式维护。 |

## Submodule

当前仓库包含以下子模块：

- `《人工智能》课外实验（要求自学）/` -> [Python-neural-network](https://github.com/3075269612/Python-neural-network)
- `第二学年小学期项目（数据爬取）/` -> [xiaohongshu-crawler](https://github.com/3075269612/xiaohongshu-crawler)

首次克隆时建议使用：

```bash
git clone --recurse-submodules <repo-url>
```

如果已经普通克隆，可以在仓库根目录执行：

```bash
git submodule update --init --recursive
```

更新子模块到远端最新提交：

```bash
git submodule update --remote --merge
```

## 使用说明

各课程目录相互独立，运行方式以对应目录中的 `README.md`、脚本入口、配置文件或源码注释为准。Python 项目通常需要先创建虚拟环境并安装依赖；C/C++ 项目通常可从 `Makefile`、`run.sh` 或项目说明中查看构建方式。

仓库中可能包含实验输出、报告成品、截图、课程资料和中间结果。若只需要代码，可优先查看各目录下的 `src/`、`scripts/`、`configs/`、`requirements.txt`、`pyproject.toml` 或 `Makefile`。

## 维护约定

- 大型、独立或可复用项目优先以 submodule 接入，主仓库只记录固定提交。
- 实验报告优先保留 Markdown 源文档，并将图片、数据和最终导出文件分目录管理。
- 不提交个人密钥、Token、Cookie、账号密码和不可公开数据。
- 课程作业内容仅用于学习记录与复盘，请遵守课程要求和学术诚信规范。

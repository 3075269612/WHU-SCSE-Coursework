# WHU-SCSE-Coursework

这是武汉大学国家网络安全学院（网络空间安全）相关课程中整理的实验与项目代码仓库。内容按课程或项目独立成目录，尽量保留可运行代码、必要数据、实验资料与阶段性输出，方便后续复盘和查阅。

## 目录概览

- `《人工智能》课内实验/`：人工智能课程课堂实验，包含搜索、约束推理、智能优化、分类、降维与聚类等主题。
- `《人工智能》课外实验（要求自学）/`：人工智能课外自学实验，当前以 Git 子模块形式维护。
- `《人工智能》期末大作业/`：人工智能期末项目，包含短信垃圾邮件分类实验代码、数据处理、结果图表与报告。
- `《密码学引论》实验大作业/`：密码学引论课程大作业，实现基于 SM4-CBC-CTS、RSA-OAEP、GTK 3 与 OpenSSL 的文件加密系统。
- `《程序设计》实验大作业/`：程序设计课程大作业相关代码与文档。
- `第一学年小学期项目/`：第一学年小学期项目代码与配套资源。

## 使用说明

各课程目录相互独立，运行方式以对应目录中的说明或源代码注释为准。Python 实验通常需要在本地创建虚拟环境并安装脚本中使用到的依赖，例如 `numpy`、`scikit-learn`、`matplotlib`、`pandas`、`opencv-python` 等。

克隆含子模块内容时可使用：

```bash
git clone --recurse-submodules <repo-url>
```

如果已经普通克隆，可在仓库根目录执行：

```bash
git submodule update --init --recursive
```
